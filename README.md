# 🚌 Rutas Bonificadas de Transporte

Una aplicación moderna e intuitiva para consultar las rutas de transporte público bonificadas, organizada por provincias y localidades.

## 🎯 Objetivo del Proyecto
El objetivo principal de esta aplicación es facilitar a los ciudadanos la búsqueda y visualización de las rutas de autobús que cuentan con bonificaciones. El sistema centraliza una gran base de datos de rutas, operadores y tipos de servicio en una interfaz ágil y fácil de usar, permitiendo a cualquier usuario encontrar su medio de transporte en segundos.

## 🛠 ¿Para qué sirve?
Esta herramienta permite:
- **Consultar Rutas**: Visualizar todas las conexiones de transporte disponibles desde una localidad específica.
- **Identificar Operadores**: Conocer qué empresa (operador) realiza cada trayecto.
- **Conocer el Tipo de Servicio**: Diferenciar rápidamente entre servicios **Regulares**, **A Demanda** o de **Prestación Conjunta**.
- **Ahorro de Tiempo**: Evitar la búsqueda en largos listados o PDFs complejos, ofreciendo una navegación jerárquica clara.

## 🚀 ¿Cómo funciona?
La aplicación utiliza un flujo de navegación diseñado para la máxima eficiencia:

1.  **Exploración por Provincias**: Las rutas están agrupadas por provincias (Ávila, León, Zamora, etc.).
2.  **Índice Alfabético**: Dentro de cada provincia, las localidades se organizan alfabéticamente mediante un selector rápido.
3.  **Buscador Inteligente**: Permite filtrar instantáneamente por nombre de pueblo, ruta u operador. Es "insensible" a acentos para que la búsqueda sea más cómoda.
4.  **Filtros Avanzados**: Permite reducir los resultados seleccionando una empresa de autobús específica o un tipo de servicio concreto (ej. solo servicios a demanda).
5.  **Tarjetas Detalladas**: Cada localidad despliega una lista de rutas con etiquetas de colores para identificar el tipo de servicio y el operador.

## ✨ Características Técnicas
- **Interfaz Premium**: Diseño basado en *Glassmorphism* con modo oscuro, degradados dinámicos y micro-animaciones.
- **React + Vite**: Desarrollado con las últimas tecnologías para una carga instantánea y fluidez total.
- **Responsivo**: Adaptado perfectamente para su uso en ordenadores, tablets y teléfonos móviles.
- **Filtros Unificados**: Lógica de filtrado inteligente que combina múltiples criterios simultáneamente.

## 💻 Instalación y Desarrollo
Si deseas ejecutar este proyecto en local:

1.  Clona el repositorio.
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Arranca el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Abre `http://localhost:5173` en tu navegador.
